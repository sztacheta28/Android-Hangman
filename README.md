# Wisielec

## 1. Opis
Klasyczny wisielec. Wychodząc do menu w trakcie odgadywania hasła możesz kontynuować grę, ale tracisz jeden punkt. W każdej chwili możesz wykorzystać podpowiedź. Przy każdym haśle wyświetla się jego kategoria.

## 2. Użyte biblioteki/narzędzia
[ButterKnife](http://jakewharton.github.io/butterknife/)  
[Android View Animations](https://github.com/daimajia/AndroidViewAnimations)

## 3. Do pobrania
